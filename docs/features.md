| B-08 | 도란도란 | 인기글 ✔️ | 좋아요·댓글 수·조회수 기반 가중치 집계로 인기글 선정. 별도 탭 또는 배지로 표시 |

### 🤖 도(道)와주세요 — AI 모델 (여민엽)

| ID | 대메뉴 | 중메뉴 | 상세 설명 |
|----|--------|--------|-----------|
| F-01 | 데이터 입력 | 파일 업로드 ✔️ | 로컬 시스템의 영상(MP4, AVI 등) 및 이미지 파일을 선택하여 서버로 전송하고 분석 대기열에 추가 |
| F-02 | 데이터 입력 | 업로드 상태 관리 ✔️ | 파일 전송률 및 AI 모델의 프레임별 처리 진행 상황을 프로그레스 바(Progress Bar)로 실시간 시각화 |
| F-03 | 실시간 분석 | 영상 미리보기 ✔️ | 분석 중인 원본 영상 또는 객체 탐지 바운딩박스(Bounding Box)가 렌더링된 가공 영상을 실시간으로 출력 |
| F-04 | 실시간 분석 | 탐지 객체 리스트 ✔️ | 영상 내에서 감지된 객체(고라니, 멧돼지, 너구리 등)의 종류를 텍스트 리스트 형태로 실시간 갱신 |
| F-05 | 로그 관리 | 탐지 시간 기록 ✔️ | 특정 객체가 감지된 시점(Timestamp)을 타임라인 형식으로 기록 |
| F-06 | 통계/시각화 | 탐지 통계 그래프 ✔️ | 분석된 데이터를 기반으로 객체별 누적 탐지 빈도수를 막대 그래프(Bar Chart)로 시각화 |
| F-07 | 결과 활용 | 결과 다운로드 ✔️ | 분석 완료 후 탐지 결과 데이터 또는 최종 분석 리포트를 로컬 저장소로 다운로드 지원 |
