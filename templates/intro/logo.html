BI Design Concept

도(道)서관의 정체성을 담은 브랜드 아이덴티티(Brand Identity)

Brand Mission

도(道)서관의 로고는 안전한 주행과 체계적인 정보 공유라는 두 가지 핵심 가치를 시각화했습니다.
운전자의 손길과 기술의 보호가 만나는 접점을 현대적인 심볼로 표현했습니다.

Symbol 01

Steering Wheel & Shield

운전자의 손길이 닿는 스티어링 휠(Steering Wheel)과 위험으로부터 보호하는
방패의 형상을 결합하여, 안전한 주행을 지향하는 '도(道)서관'의 핵심 가치를 담았습니다.

Symbol 02

Road & Archive

로고 중앙을 관통하는 도로와 이를 감싸는 책(도서)의 실루엣은
도로 위의 지식과 정보를 체계적으로 관리하는 정보 창고의 의미를 상징합니다.

Color Palette

메인 컬러인 Teal은 스마트한 AI 기술력과 심리적 안전감을 상징하며,
Dark Grey를 서브 컬러로 매치하여 서비스의 신뢰도와 무게감을 더했습니다.

Main Color : Teal Hex Code : #20828A
Sub Color : Dark Grey Hex Code : #212529
